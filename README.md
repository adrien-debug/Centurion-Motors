# Centurion Motors - Outil de Gestion

Application Next.js pour la gestion d'un garage de voitures de luxe basé à Cannes.

## Fonctionnalités

- **Dashboard** : Vue d'ensemble du marché et des opportunités
- **Scanner de Marché** : Scan des sites européens pour trouver les meilleures affaires
- **Recherche Véhicule** : Estimation de prix par référence de véhicule
- **Calculateur de Prix** : Calcul du prix total avec options
- **Analytics** : Statistiques et tendances du marché

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Build

```bash
npm run build
npm start
```

## Technologies

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Recharts (graphiques)
- Lucide React (icônes)
